# 📊 Data Profession Survey Dashboard

An interactive Power BI dashboard visualizing insights from a survey of data professionals — covering salaries, job satisfaction, programming language preferences, and demographics.

---

## 📋 Project Overview

This project transforms raw survey data from data professionals into a compelling Power BI dashboard. It explores who works in data, what they earn, what tools they love, and how happy they are in their roles — broken down by job title, country, and gender.

---

## 🖥️ Dashboard Page

### Page 1 — Survey Insights

| Visual Type | What It Shows |
|---|---|
| **Card** | Total Survey Takers |
| **Card** | Average Age of Survey Respondents |
| **Bar Chart** | Average Salary by Job Title |
| **Treemap** | Distribution of Survey Takers by Country |
| **Column Chart** | Favourite Programming Language by Job Title |
| **Gauge** | Happiness Score — Salary Satisfaction |
| **Gauge** | Happiness Score — Work/Life Balance |
| **Donut Chart** | Gender breakdown (Male/Female) |

---

## 🗂️ Survey Questions Captured

| Field | Description |
|---|---|
| `Q1 - Which Title Best Fits your Current Role?` | Job title / role in data |
| `Q5 - Favorite Programming Language` | Preferred coding language |
| `Q6 - Salary Happiness` | Satisfaction score for current salary |
| `Q6 - Work/Life Balance Happiness` | Satisfaction score for work/life balance |
| `Q9 - Male/Female?` | Gender of respondent |
| `Q10 - Current Age` | Age of respondent |
| `Q11 - Which Country do you live in?` | Country of residence |
| `Unique ID` | Respondent identifier |

---

## 🔍 Key Insights

- **Salary vs. satisfaction** — gauge visuals reveal how well compensation aligns with expectations across roles
- **Top programming languages** vary meaningfully by job title, reflecting tool preferences across data analysts, engineers, and scientists
- **Country distribution** via treemap shows where the data professional community is most concentrated
- **Work/life balance scores** sit alongside salary happiness, enabling side-by-side quality-of-life comparison

---

## 🛠️ Tools Used

- **Power BI Desktop** — Data modeling, DAX measures, and interactive dashboard design
- **DAX** — Calculated measures for averages, counts, and KPI cards
- **Power Query** — Data transformation and cleaning of raw survey responses

---

## 🚀 How to Use

1. Download `Data_Profession_Survey.pbix`
2. Open in **Power BI Desktop** (free download at [powerbi.microsoft.com](https://powerbi.microsoft.com))
3. Navigate to **Page 1** to explore the full dashboard
4. Click on any visual to cross-filter and drill into specific job titles, countries, or languages
